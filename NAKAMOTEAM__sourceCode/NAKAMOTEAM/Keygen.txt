CERT
% Per ogni server vengono generati, usando keytool, una coppia di chiavi PK-SK ed un certificato, che poi vengono messi in un file che prende il nome di keystore.
% Ogni server dispone di un proprio keystore
% Per ogni client vengono generati una coppia di chiavi PK-SK ed un certificato con OpenSSL

SGen
./keytool -genkey -noprompt -trustcacerts -keyalg RSA -alias gencert -dname "cn=sgen, ou=CEN, l=Campania" -keypass sergen -keystore keystoreGen.jks
./keytool -export -alias gencert -storepass sergen -file keystoreGen.cer -keystore keystoreGen.jks
